## 🚀SOBRE EL PROYECTO

**Misa Bot** es una infraestructura de mensajería automatizada de alto rendimiento. Desarrollado desde cero con **Node.js** y **Baileys**, este sistema permite la gestión de múltiples sub-bots (sockets) de manera simultánea, ofreciendo estabilidad y una interfaz estética en consola.

-----

## 🛠️ INSTALACIÓN Y ACTUALIZACIÓN

### ☁️ Pterodactyl Panel

1.  Sube los archivos al panel.
2.  El panel instalará las dependencias automáticamente al dar a **START**.
3.  Vincula tu número siguiendo las instrucciones de la consola.

### 📱 Termux / VPS / Actualizar

1.  Dar permisos de almacenamiento (Acepta el mensaje que saldrá en pantalla)

<!-- end list -->

```bash
termux-setup-storage
```

2.  Actualizar los paquetes del sistema

<!-- end list -->

```bash
pkg update && pkg upgrade -y
```

3.  Instalar las herramientas necesarias (Git y Node.js)

<!-- end list -->

```bash
pkg install git nodejs ffmpeg imagemagick -y
```

4.  Clonar el repositorio git

<!-- end list -->

```bash
git clone https://github.com/yannielmedrano1-sys/Misa-Bot
```

5.  Entrar a la carpeta del proyecto

<!-- end list -->

```bash
cd Misa-Bot
```

6.  Instalar las dependencias del bot

<!-- end list -->

```bash
npm install
```

7.  Iniciar el bot por primera vez

<!-- end list -->

```bash
npm start
```
